# 思源笔记看板插件

一个简单的看板插件，帮助你在思源笔记中可视化管理任务。

## 功能特性

- **默认列**：四个默认列 - 待办、进行中、已完成、阻塞
- **自定义列**：根据需要添加、重命名和删除列
- **任务管理**：创建和编辑任务，包含标题、描述、截止日期
- **拖拽排序**：轻松在列之间移动任务
- **截止日期可视化**：颜色编码的截止日期标签（正常、临近、紧急、逾期）
- **数据持久化**：所有数据自动保存
- **主题支持**：支持亮色和暗色主题

## 安装方法

1. 将此仓库克隆到你的思源笔记插件目录：
   ```
   {工作空间}/data/plugins/siyuan-plugin-kanban
   ```

2. 或者下载发布包并解压到插件目录

3. 在思源笔记中启用插件

## 使用方法

1. 点击顶部工具栏的看板图标打开看板
2. 点击任意列中的"+ 添加任务"创建新任务
3. 点击任务卡片编辑任务
4. 拖拽任务在列之间移动
5. 使用列标题按钮编辑或删除列
6. 使用"添加列"添加新列
7. 使用"重置看板"恢复默认列（会清除所有数据）

## 任务属性

- **标题**：任务标题（必填）
- **描述**：任务描述（可选，支持多行文本）
- **截止日期**：截止日期（可选，颜色编码显示）
- **创建时间**：自动生成的时间戳

## 快捷键

- `⌘⇧K` (Mac) / `Ctrl⇧K` (Windows)：打开看板

## 数据存储

数据存储在 `data/plugins/siyuan-plugin-kanban/kanban.json`
