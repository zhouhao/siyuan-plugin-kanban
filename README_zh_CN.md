# 思源笔记看板插件

一个简单的看板插件，帮助你在思源笔记中可视化管理任务。支持多个独立看板。

## 功能特性

- **多看板管理**：创建、重命名和删除独立的看板
- **侧边栏面板**：通过 Dock 侧边栏管理所有看板
- **标签页集成**：每个看板在独立的思源标签页中打开
- **默认列**：四个默认列 - 待办、进行中、阻塞、已完成
- **自定义列**：根据需要添加、重命名和删除列
- **任务管理**：创建、编辑和删除任务，包含标题、描述、截止日期
- **拖拽排序**：通过拖拽在列之间移动任务
- **完成状态**："已完成"列中的任务以删除线样式显示
- **截止日期可视化**：颜色编码的截止日期标签（正常、临近、紧急、逾期）
- **数据持久化**：所有数据自动保存
- **主题支持**：支持亮色和暗色主题

## 安装方法

1. 将此仓库克隆到你的思源笔记插件目录：
   ```
   {工作空间}/data/plugins/siyuan-plugin-kanban
   ```

2. 或者下载发布包并解压到插件目录

3. 在思源笔记中启用插件

## 使用方法

### 看板管理

- 点击顶部工具栏的**看板图标**，弹出下拉菜单列出所有看板
- 选择一个看板在新标签页中打开，或点击**新建看板**创建新的看板
- 使用**侧边栏面板**（左下角 Dock）浏览、重命名或删除看板

### 使用看板

1. 点击任意列中的"+ 添加任务"创建新任务
2. 点击任务卡片编辑任务
3. 拖拽任务在列之间移动
4. 使用列标题按钮编辑或删除列
5. 使用"添加列"添加新列
6. 使用"重置看板"恢复默认列（会清除该看板所有任务）

## 任务属性

- **标题**：任务标题（必填）
- **描述**：任务描述（可选，支持多行文本）
- **截止日期**：截止日期（可选，颜色编码显示）
- **创建时间**：自动生成的时间戳

## 数据存储

每个看板独立存储在 `data/storage/petal/siyuan-plugin-kanban/` 下：

- `boards-index` - 所有看板的索引（名称、ID）
- `board-{id}` - 单个看板数据（列和任务）

旧版单看板格式（`kanban`）的数据会在首次加载时自动迁移。
